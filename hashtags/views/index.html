About

This web app displays a report of most common 10 keywords in individual documents or over a selection of them. The data is stored in a sqlite database called hashtags.db.

To process more local files, please place them in the /documents directory and they'll be loaded upon refresh. The application supports the following formats: txt, pdf, html and web pages.
